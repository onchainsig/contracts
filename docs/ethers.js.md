# ethers.js

`ethers.js` 是一个完整紧凑的开源库，用于与 EVM 兼容链及其生态进行交互，相比于 `web3.js` 更好一些：代码库小，更安全，支持 ENS 等；所以，首推使用 `ethers.js` .

目前有 v5 和 v6 两个版本，新项目推荐使用 v6 .
